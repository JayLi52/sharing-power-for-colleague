# 共享充电小程序
项目描述：毕业设计项目前端，相比于开发APP应用小程序更容易让用户接受，相比于网页小程序具有微信的端能力有比网页更加丰富的功能。
  >> - 使用小程序 Map 组件渲染机柜地点
  >> - 支持展示附近机柜并在地图展示，提供地图导航
  >> - 微信扫机柜充电宝，生成订单
  >> - 获取用户信息并从后端获取使用记录
