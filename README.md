# 共享充电小程序

微信小程序的开发主要参考自微信官方文档，除了一些基础的 SDK 能力，还使用了一些媒体能力：地图、图片；数据缓存 localStorage：永久存放与用户相关的历史记录；位置获取务：定位本机所在位置；canvas 画布： 绘制充电宝弹出加载动画等。
